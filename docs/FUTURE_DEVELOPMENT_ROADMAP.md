# Future Development Roadmap

Future phases are defined in `PROJECT_BRIEF.md`. This file intentionally contains no implementation claims for work not yet completed.

## Next: automatic music analysis

- Automatic BPM and beat detection belong to the next phase, not Phase 2A.
- Automatic music analysis must always retain manual calibration and the full manual workflow as a reliable fallback.
- The formal choreography start must always be confirmed by the user; the system must never decide it automatically.

## Future experience improvements

- Natural count voice and prerecorded count voice remain future options; Phase 2A.1 intentionally contains no speech runtime.
- High-precision count/audio synchronization is deferred until a later dedicated audio design.
- Automatic BPM and beat analysis remain a later phase and must preserve manual calibration as fallback.
- Picture brightness and contrast enhancement remain a non-blocking future display improvement.
