## 当前主要功能

- 在浏览器中选择本地 MP4、MOV 或 WebM 舞蹈视频，进行慢放、镜像、分段、循环、全屏和教师画面缩放/拖动。
- 开启本机摄像头进行教师与学习者分屏对比。
- 主动录制分屏练习，选择 3、5 或 10 秒倒计时，在本地回看、变速回看、下载 WebM、删除或再录一次。
- 第1段自动带上系统计算的预备拍；默认按音乐与动作起点自动分段。“自定义分段”默认折叠，打开后可拖动主进度条并确认单个分界。
- 教师画面统一使用镜像、缩小、放大、重置和全屏；重置恢复完整显示、默认大小和居中位置。

## 隐私与保存

- 舞蹈视频、摄像头画面和生成的练习录像不会上传。
- 录像不会请求或录制麦克风、环境声音；浏览器支持时会包含教师视频声音，否则可能只生成画面。
- 录像先保存在当前浏览器页面的内存中，不会自动永久保存。刷新或关闭页面前，如需保留，必须由用户主动下载 WebM。
- 当前没有用户账号、云端同步、云端录像保存或动作识别。

## 启动方式

在项目根目录运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\start_frontend.ps1
```

然后打开 PowerShell 实际显示的地址，通常是 <http://127.0.0.1:5173/>。
